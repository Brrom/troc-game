Bienvenue sur le site Trocgame, le site de troc de jeux entre particuliers.

Nouvel utilisateur?

Si vous n'êtes pas sûr de vouloir vous lancer dans le troc, pas de problème, prenez le temps de parcourir notre site et les jeux à disposition. Seul le troc nécessite de s'inscrire.
Rendez-vous sur la page 'Recherche' pour trouver les jeux qui vous intéressent en cherchant leur titre. Vous pouvez aussi découvrir de nouveaux jeux en faisant une recherche par catégorie, âge, nombre de joueurs ou durée moyenne d'une partie.


Envie de troquer?

Vous avez trouvé LE jeu qui vous plaît? C'est le moment idéal pour vous rendre sur la page 'Inscription' afin de créer votre compte d'utilisateur. Il vous suffit de rentrer quelques renseignements sur vous-même et vous voilà prêt à échanger avec d'autres utilisateurs.
Pour cela, il vous suffit, sur la page de 'Détails du jeu' de cliquer sur le bouton 'Je souhaite ce jeu'. Vous avez alors accès aux coordonnées de l'utilisateur ayant mis le jeu à disposition. À vous de jouer pour le convaincre de vous troquer son jeu. Bon courage et bon jeu!


Et si vous proposiez vos jeux?

Vous souhaitez proposer un ou plusieurs jeux au troc? Rien de plus simple, rendez-vous sur votre compte. Cliquez sur 'Ajouter un jeu à troquer' et c'est parti.
Il vous suffit d'entrer quelques renseignements sur votre jeu tels que son titre, sa catégorie, le nombre de joueurs, … Tous les champs avec une astérisque (*) sont obligatoires afin de pouvoir créer la fiche de votre jeu et avoir ainsi une chance qu'il soit trouvé par les autres utilisateurs lors de leurs recherches.
